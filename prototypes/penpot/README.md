# Jakub Olša Penpot Prototype

Tento priečinok obsahuje editovateľný SVG prototyp pre Penpot.

## Import

1. Otvor Penpot.
2. Vytvor nový file alebo project.
3. Importni `jakub-home-penpot-prototype.svg`.
4. Po importe si rozklikni skupiny/artboardy:
   - `Desktop / hero`
   - `Desktop / menu open`
   - `Mobile / hero`

## Poznámka

Toto nie je screenshot produkčnej stránky. Je to návrhový prototyp pre rýchle posúvanie hero sekcie, top baru a menu. Obrázok Jakuba je zatiaľ riešený ako placeholder, aby sa layout dal v Penpote ľahko preskladať.
